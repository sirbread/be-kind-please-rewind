## Key UI Improvements

1. **Three new buttons** in the main toolbar for file and snapshot management
2. **Context-sensitive button states** - remove and manage buttons only enabled when file is selected
3. **Comprehensive snapshot management** in a dedicated dialog window
4. **Flexible deletion criteria** for managing storage space
5. **Import/export functionality** for backup and migration
6. **Notes integration** throughout the UI with inline editing
7. **Clear confirmation dialogs** for destructive operations
